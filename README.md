# Autonomous-Financial-Research-Analyst

Built an Autonomous Financial Research Analyst agent using LangChain and LangGraph. The system reduces investment research time from 4–6 hours per company to under 10 minutes by autonomously gathering real-time stock prices, 3-year historical performance, and financial news sentiment. Powered by GPT-4o-mini with 4 specialized tools — Yahoo Finance, Tavily Search, and sentiment analysis — the agent independently selects tools, handles errors, and delivers structured investment recommendations without human intervention
